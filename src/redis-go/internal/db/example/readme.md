#### 核心功能说明（新增部分）
|方法名|	功能描述|	原子性|	兼容场景|
|---- |----|----|----|
|SetNX|	键不存在时设置，返回设置结果|	是|	分布式锁、唯一资源创建|
|SetXX	|键存在时更新，返回更新结果|	是|	仅更新已有数据，防止误创建|
|MSet	|批量设置多键值对，全部生效或全部不生效	|是|	批量初始化、数据同步|
|MSetNX	|所有键不存在才批量设置，返回设置结果|	是|	批量创建唯一数据集合|
|GetSet	|原子替换：设置新值并返回旧值，键不存在时也会设置新值（兼容 Redis 行为）	|是|	统计计数更新、原子替换场景|
|MGet|	批量获取多键值对，返回顺序与传入键一致，不存在的键返回空字符串|	是	|批量查询，减少锁竞争|
|GetRange|	截取字符串子串，支持正负索引，兼容多字节字符（如中文）|	是	|手机号脱敏、内容截取|
|StrLen	|获取字符串的字符长度（非字节长度），支持多字节字符	|是	|长度校验、内容统计|

#### 核心完善功能说明（重点 HDel 系列）
#### 核心新增功能说明
|方法名	|功能描述|	适用场景|
|----|----|----|
|HSetNX	|字段不存在时才设置（原子操作），避免覆盖已有值	|初始化唯一字段、分布式锁场景|
|HSetMulti|	批量设置多个字段，减少遍历开销，效率更高	|批量初始化 / 更新哈希表|
|HIncrBy	|数值字段自增 / 自减（支持 int/int64 / 字符串数字），字段不存在则初始化为增量	|计数器、积分统计、库存变更|
|HMGet|	批量获取多个字段，返回顺序与传入字段一致	|批量查询，减少循环调用|
|HKeys	|获取哈希表所有字段名	|遍历字段、字段校验|
|HVals|	获取哈希表所有字段值	|数据汇总、统计分析|
|HDelMulti|	批量删除多个字段，返回成功删除数量	|批量清理无效字段|
|HClear	|清空指定哈希表的所有字段，返回清空数量	|重置哈希表数据|
|HStrLen	|获取字段值的字符串长度（支持任意类型值）|	长度校验、字符串统计|

|方法名	| 功能描述 |	特性亮点|
|----|----|----|
|HDel|	单字段删除，返回删除结果	|线程安全、时间复杂度 O (1)（平均）|
|HDelMulti|	批量删除多个字段，返回成功删除数量	|原子操作、优化遍历逻辑、避免重复删除|
|HClear|	清空指定哈希表的所有字段，返回清空数量	|原子操作、高效遍历、不影响其他哈希表|
|HDelHash|	删除整个哈希表（含所有字段），返回是否存在	|原子操作、支持哈希表级别的删除|
|HashExists|	检查指定哈希表是否存在（至少有一个字段）|	线程安全、快速判断|
|HashCount|	获取当前所有独立的哈希表数量	|统计哈希表整体规模|
