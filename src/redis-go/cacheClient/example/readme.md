
#### 核心新增功能说明
|命令|	功能描述	|适用场景|
|  ----  | ----  | ----  |
|SetNX	|键不存在时设置（原子）|	分布式锁、避免重复创建|
|SetXX	|键存在时更新（原子）	|仅更新已有数据，防止误创建|
|MSet	|批量设置多键值对（原子）|	批量初始化数据|
|MSetNX|	批量设置（所有键不存在才生效，原子）|	批量创建唯一数据|
|GetSet	|设置新值并返回旧值（原子）|	原子替换、统计计数更新|
|MGet|	批量获取多键值对（原子）|	批量查询，减少网络往返|
|GetRange|	获取字符串子串|	截取部分内容（如手机号脱敏）|
|StrLen|	获取字符串长度	|校验值长度、统计字符数|

#### 核心新增哈希命令说明
|命令	|功能描述|	适用场景|
|  ----  | ----  | ----  |
|HSetNX|	哈希字段不存在时设置（原子）|	避免覆盖已有字段值|
|HDel|	批量删除哈希字段	|清理无用字段|
|HMGet	|批量获取多字段（原子）|	减少网络往返，批量查询|
|HGetAll|	获取所有字段和值|	导出整个哈希数据|
|HKeys	|获取所有字段名	|遍历哈希字段|
|HVals	|获取所有字段值	|汇总哈希值集合|
|HLen|	获取字段总数|	统计哈希大小|
|HExists|	判断字段是否存在|	避免操作不存在的字段|
